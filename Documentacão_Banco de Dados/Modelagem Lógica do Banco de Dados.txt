Modelagem Lógica do Banco de Dados

Tabela: Produto
* produto_id (Chave Primária, Inteiro)
* descricao (Texto)
* preco_unitario (Decimal)
* disponibilidade (Inteiro)
* qtd_estoque (Inteiro)
* image_produto (Texto)
* largura (Ponto Flutuante)
* altura (Ponto Flutuante)
* profundidade (Ponto Flutuante)
* peso (Ponto Flutuante)
* funcionalidade_adicionais (Texto, Opcional)
* instrucoes_uso (Texto, Opcional)
* categoria_produto (Texto)



Tabela: Comprador
* comprador_id (Chave Primária, Inteiro)
* nome_completo (Texto)
* cpf (Texto)
* endereco (Texto)
* cep (Texto)
* cidade (Texto)
* estado (Texto)
* telefone (Texto)
* email (Texto)

Tabela: Vendedor
* vendedor_id (Chave Primária, Inteiro)
* nome_completo (Texto)
* cnpj (Texto)
* endereco (Texto)
* cep (Texto)
* cidade (Texto)
* estado (Texto)
* email (Texto)
* telefone (Texto)
* nome_loja (Texto)
* descricao_loja (Texto)
* logo_loja (Texto)
* numero_conta (Texto)
* banco (Texto)
* tipo_da_conta (Texto)
* metodo_pagamento_preferencial (Texto)

Tabela: Representante Suporte Cliente
* representante_id (Chave Primária, Inteiro)
* treinamento_acessibilidade (Texto)
* canais_comunicacao (Inteiro)
* registro_interacao (Texto, Opcional)
* vendedor_id (Inteiro)

Tabela: Vendedor Autenticação
* vendedor_id (Chave Primária, Inteiro)
* nome_usuario (Texto)
* senha (Texto, Opcional)

Tabela: Preferências Comprador
* preferencial_id (Chave Primária, Inteiro)
* produto_id (Inteiro)
* comprador_id (Inteiro)
* historico_compras (Texto)
* preferencias_produtos (Texto, Opcional)
* prefencias_valores (Ponto Flutuante, Opcional)

Tabela: Comprador Autenticação
* comprador_id (Chave Primária, Inteiro)
* nome_usuario (Texto)
* senha (Texto)

Tabela: Avaliação
* avaliacao_id (Chave Primária, Inteiro)
* comprador_id (Inteiro)
* vendedor_id (Inteiro)
* produto_id (Inteiro)
* pontuacao (Decimal)
* comentario (Texto, Opcional)
* data_avaliacao (Data e Hora, Opcional)

Tabela: Carrinho
* carrinho_id (Chave Primária, Inteiro)
* comprador_id (Inteiro)
* produto_id (Inteiro)
